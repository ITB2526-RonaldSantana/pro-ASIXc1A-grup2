# Triggers

Aquest document descriu els triggers implementats per a la base de dades. No hi ha captures específiques per a aquesta pàgina en la carpeta `capturas/07-bd/`.

## Descripció

- Els triggers s'utilitzen per automatitzar comprovacions i insercions addicionals després d'operacions `INSERT`, `UPDATE` o `DELETE`.
- Són útils per mantenir la integritat de la informació sense necessitat d'aplicacions externes.

## Resum

Tot i no tenir captures, aquest document completa el bloc de la base de dades i permet organitzar millor l'apartat 07.
