# Rols i permisos

Aquest document descriu el model de rols i permisos de la base de dades. No hi ha captures específiques per a aquesta pàgina en la carpeta `capturas/07-bd/`.

## Descripció

- Els rols es defineixen per controlar l'accés a les taules i operacions de la base de dades.
- Els permisos s'apliquen amb `GRANT` i es revoquen amb `REVOKE`.

## Resum

Aquest apartat no conté captures, però forma part del conjunt documental del projecte que explica la seguretat i el control d'accés.
